## Key Learnings & Gotchas

- **Backslash + inline comment breaks shell continuation.** In markdown code blocks showing multi-line shell commands, `\   # comment` makes the `\` escape a space instead of continuing the line. Always put comments on their own line above the command, or remove them. pr-agent caught this on round 1.
- **Prose and code must be internally consistent.** The intro said "no API-key domain restrictions" but the masking section below contradicted it. Reviewer caught on round 2. When documenting a mode that eliminates *most* noise, explicitly carve out the exceptions rather than claiming total elimination.
- **`gh pr merge --delete-branch` in a worktree** errors on local branch deletion (the branch is in use by the worktree) but the GitHub-side merge and remote branch deletion succeed. Ignore the local error; verify with `gh pr view`.
