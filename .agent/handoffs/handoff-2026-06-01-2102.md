## Key Learnings & Gotchas
- pr-agent hallucinated `capture_urls.py` — the actual VR skill uses `crawl_urls.py` → `capture.py` → `diff_report.py`. Always verify script names against the actual file system before accepting pr-agent script name suggestions.
- pr-agent's ticket compliance check doesn't handle repo-relative issue references well (the sub-issues fetch errors out with `'NoneType' object has no attribute 'get'`).
- The `code-quality.md` file lives at `skills/silverstripe-version-upgrade/references/`, so cross-references to sibling skills need `../../` not `../`.
- Allowlist `.gitignore` (`/*` then `!/path/`) is the recommended pattern when `.agent/` needs tracking.
